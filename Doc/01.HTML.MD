## La Structure de base d'une page HTML5
```html
<!DOCTYPE html>

<html>
 <head>
  <meta charset="utf-8" />
  <title>Titre</title>
  <link rel="stylesheet" href="style.css" />
 </head>

 <header>
 </header>

 <body>
 </body>

</html>
```

-------------------------------------------------------------------------------------------------------------------------
### Le doctype (HML5)
```html
<!DOCTYPE html>
```
-------------------------------------------------------------------------------------------------------------------------------------------------
### La balise HTML
Elle englobe tout le contenu de votre page. Comme vous pouvez le voir, la balise fermante </html> se trouve tout à la fin du code !
```html
<html>
 ...
</html>
```

## Les liens

#### Un lien vers un autre site
```html
<a href="https://openclassrooms.com">OpenClassrooms</a>
```
#### Un lien vers une ancre
```html
<h2 id="mon_ancre">Titre</h2>
<a href="#mon_ancre">Aller vers l'ancre</a>
```
#### Lien vers une ancre située dans une autre page
```html
<a href="ancres.html#rollers"></a>
```
#### Un lien qui ouvre une nouvelle fenêtre 
```html
<a href="ancres.html#rollers" title="TEST" target="_blank">ICI</a>
```
#### Un lien pour envoyer un e-mail
```html
<a href="mailto:votrenom@bidule.com">Envoyez-moi un e-mail !</a>
```
#### Un lien pour télécharger un fichier
```html
<a href="monfichier.zip">Télécharger le fichier</a>
```

## Les Images

#### Insertion d'une image
```html
<img src="images/montagne.jpg" alt="Photo de montagne" />
```
#### Ajouter une infobulle
```html
<img src="images/montagne.jpg" alt="Photo de montagne" title="C'est beau les Alpes quand même !" />
```
#### Miniature cliquable
```html
<a href="img/montagne.jpg"><img src="img/montagne_mini.jpg" alt="Photo de montagne" title="Cliquez pour agrandir" /></a>
```
-------------------------------------------------------------------------------------------------------------------------
#### Balise de type Inline (span, strong, em)
```html
<span> </span> : C'est une balise que l'on place au sein d'un paragraphe de texte.

<p>Bonjour et <span class="salutations">bienvenue</span> sur mon site !</p>
```
-------------------------------------------------------------------------------------------------------------------------
#### Balise de type Block (div, p, h1 ..)
```html
<div> </div>  : c'est une balise qui entoure un bloc de texte.
```
