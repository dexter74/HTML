### Les balises

#### Paires
```html
<titre>Ceci est un titre</titre>
```

#### Orphelines
```html
<image />
```

### Les attributs
```html

```

-------------------------------------------------------------------------------------------------------------------------
## La Structure de base d'une page HTML5
```html
<!DOCTYPE html>
<html>
    <head>
        <!-- En-tête de la page -->
        <meta charset="utf-8" />
        <title>Titre</title>
    </head>

    <body>
        <!-- Corps de la page -->
    </body>
</html>
```

-------------------------------------------------------------------------------------------------------------------------
### Le doctype (HML5)
```html
<!DOCTYPE html>
```

-------------------------------------------------------------------------------------------------------------------------
### La balise HTML
C'est la balise principale du code. Elle englobe tout le contenu de votre page. Comme vous pouvez le voir, la balise fermante </html>  se trouve tout à la fin du code !

```html
<html>
</html>
```
