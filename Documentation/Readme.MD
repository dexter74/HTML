## La Structure de base d'une page HTML5
```html
<!DOCTYPE html>
<html>
    <head>
        <!-- En-tête de la page -->
        <meta charset="utf-8" />
        <title>Titre</title>
    </head>

    <body>
        <!-- Corps de la page -->
    </body>
</html>
```

-------------------------------------------------------------------------------------------------------------------------
### Le doctype (HML5)
```html
<!DOCTYPE html>
```

-------------------------------------------------------------------------------------------------------------------------
### La balise HTML
C'est la balise principale du code. Elle englobe tout le contenu de votre page. Comme vous pouvez le voir, la balise fermante </html>  se trouve tout à la fin du code !

```html
<html>
</html>
```

-------------------------------------------------------------------------------------------------------------------------
### L'en-tête et le Corps

#### L'en-tête
Cette section donne quelques informations générales sur la page, comme son titre, l'encodage
```html

</head>
```

#### Le Corps
```html
<body>
</body>
```

#### Les titres
```html
<h1></h1> : Titre très important.
```

## Les Listes

#### Liste non ordonnée
```html
<ul>
    <li>Fraises</li>
    <li>Framboises</li>
    <li>Cerises</li>
</ul>
```

#### Liste ordonnée
```html
<ol>
    <li>Je me lève.</li>
    <li>Je mange et je bois.</li>
    <li>Je retourne me coucher.</li>
</ol>
```
